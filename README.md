# intro-emb-coursera-c1m1
Repository for C1M1 of Introduction to Embedded Systems Software and Development Environments Coursera Course

# Compilation

## Using gcc directly

```
cd statistics
gcc stats.c -o c1m1
```

## Using cmake

```
mkdir build
cd build
cmake ..
cmake --build .
```


# Running the program

```
./c1m1
```
